## 🚀 Cómo Empezar (3 Pasos)

### Paso 1: Descargar y Extraer
```bash
# Descarga cmva-redesign-complete.tar.gz
# Extrae: click derecho → Extract All
# O en terminal:
tar -xzf cmva-redesign-complete.tar.gz
```

### Paso 2: Instalar
```bash
cd cmva-redesign
npm install
```

### Paso 3: Ejecutar
```bash
npm run dev
# Abre http://localhost:3000
```

**¡Listo!** Tu sitio web está corriendo localmente 🎉

## 🎨 Personalización Rápida

### Cambiar Colores (5 minutos)
1. Abre `tailwind.config.js`
2. Busca sección `colors`
3. Reemplaza valores hex
4. ¡Listo! Se actualiza automáticamente

### Cambiar Textos (10 minutos)
1. Abre componentes en `components/`
2. Busca texto a cambiar
3. Reemplaza
4. Guarda y recarga navegador

### Agregar Logo (5 minutos)
1. Copia logo a `public/images/`
2. Edita `components/Header.tsx`
3. Reemplaza emoji con imagen
4. Listo

Más detalles en `GUIA_PERSONALIZACION.md`
