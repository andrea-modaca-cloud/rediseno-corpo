# Proyecto: Rediseño Sitio Web Corporativo (CMVA)

## 📋 Descripción
Este proyecto tiene como objetivo modernizar la presencia digital de la Corporación Municipal de Villa Alemana (CMVA), mejorando la experiencia de usuario (UX), la estética visual (UI) y la accesibilidad de la información para los ciudadanos.

## 🎯 Objetivos Principales
- Modernización de la interfaz visual.
- Mejora en la navegación y acceso a servicios municipales.
- Diseño responsivo (móvil y escritorio).
- Integración de las nuevas firmas corporativas y lineamientos de marca.
